  其实，我对你是有一些失望的 😅。当初给你定级实习生，是高于你面试时的水平的 🤗。我是希望进来后，你能够拼一把，快速成长起来的。实习生这个层级，不是把事情做好就可以的 🥱。你需要有体系化思考的能力。你做的事情，他的价值点在哪里？你是否作出了壁垒形成了核心竞争力？你做的事情，和公司内其他团队的差异化在哪里？😡
